# TBH Companion

แพลตฟอร์มผู้ช่วยสำหรับเกม **TBH: Taskbar Hero** — ดูสถานะ ฟาร์ม ของในกระเป๋า/คลัง
ฮีโร่ เพ็ท รูน และ **สั่งเริ่ม/หยุดฟาร์มจากระยะไกล** ได้ผ่านเว็บ

ประกอบด้วย 2 ส่วน:

- **เว็บ** (Next.js + Supabase, ดีพลอยบน Vercel) — หน้าจอดูข้อมูลและสั่งงาน
- **เอเจนต์** (โปรแกรม Python บนเครื่องที่เล่นเกม) — อ่านไฟล์เซฟ ส่งข้อมูลขึ้นเว็บ
  และรับคำสั่งมาควบคุมเกม

ข้อมูลทั้งหมด (กระเป๋า ทอง ด่าน อุปกรณ์) มาจากการ **ถอดรหัสไฟล์เซฟ `.es3` โดยตรง**
ไม่ได้ใช้การอ่านภาพหน้าจอ จึงแม่นยำ ส่วนรายการของที่ดรอปและกิจกรรมต่าง ๆ ได้จากการ
**เทียบไฟล์เซฟก่อน/หลัง**

## 1. เตรียม Supabase + ดีพลอยเว็บ

1. สร้างโปรเจกต์ Supabase แล้วเปิด **SQL Editor** รันไฟล์
   `supabase/migrations/0001_initial_schema.sql` (สร้างตาราง, RLS และเปิด Realtime)
2. คัดลอก `.env.example` เป็น `.env.local` แล้วใส่ค่าจริงจาก Supabase
   (Project Settings → API):
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `SUPABASE_SERVICE_ROLE_KEY` (ความลับฝั่งเซิร์ฟเวอร์ ห้ามเปิดเผย)
3. รันในเครื่อง: `pnpm install` แล้ว `pnpm dev` (เปิด http://localhost:3000)
4. ดีพลอยขึ้น **Vercel**: ใส่ตัวแปรทั้ง 3 ตัวข้างบนใน Environment Variables
   แล้วดีพลอย

## 2. สร้างเอเจนต์ในเว็บ

สมัคร/เข้าสู่ระบบ → ไปหน้า **เครื่องเอเจนต์ (/agents)** → กด “เพิ่มเครื่อง” →
**คัดลอกโทเคน** ที่ขึ้นต้นด้วย `tbh_` (จะแสดงครั้งเดียว)

## 3. ติดตั้งเอเจนต์บนเครื่องที่เล่นเกม

ต้องมี Python 3.11+ บน Windows

```powershell
cd agent
pip install -r requirements.txt
python run.py setup     # ใส่ URL เว็บ, โทเคน, ตรวจหาไฟล์เซฟอัตโนมัติ
python run.py           # เริ่มทำงาน (อ่านเซฟ ส่งข้อมูล รับคำสั่ง)
```

ตั้งค่าเพิ่มเติมได้ใน `agent/config.json`:

- `windowTitle` — ชื่อหน้าต่างเกม (บางส่วนก็ได้) สำหรับสกรีนช็อต/โฟกัส
- `controls.startFarmHotkey` / `stopFarmHotkey` — ปุ่มลัดเริ่ม/หยุดฟาร์ม
  (ตั้งให้ตรงกับที่ผูกไว้ในเกม)
- `savePath` — เว้นว่างไว้เพื่อค้นหาไฟล์เซฟอัตโนมัติที่
  `%USERPROFILE%\AppData\LocalLow\TesseractStudio\TaskbarHero\SaveFile_Live.es3`

จากนั้นเปิดเว็บค้างไว้ ข้อมูลจะอัปเดตแบบเรียลไทม์ และสั่งงานจากหน้า **ควบคุม** ได้เลย
