# TBH Companion Agent — ติดตั้งบนคอมเครื่องใหม่

> ไฟล์นี้เขียนให้ "คนหรือ AI" ทำตามได้ทีละขั้นจนจบ
> เป้าหมาย: เอาโปรแกรมเอเจนต์ไปรันบนพีซีอีกเครื่อง แล้วให้มันส่งข้อมูลเกมขึ้นเว็บเดียวกัน

---

## 0. เข้าใจภาพรวมก่อน (สำคัญ อ่าน 30 วิ)

- **เอเจนต์ = โปรแกรม Python** ที่รันบน "พีซีที่เล่นเกม" คอยอ่านไฟล์เซฟของเกมแล้วส่งขึ้นเว็บ
- **1 เครื่อง = 1 เอเจนต์ = 1 token แยกกัน** — ห้ามเอา token ของเครื่องอื่นมาใช้ซ้ำ
- **เว็บสั่งเปิดเอเจนต์จากระยะไกลไม่ได้** (ไม่มีปุ่มบนเว็บ) ต้องเปิดโปรแกรมเองบนเครื่องนั้น
  สถานะ `online/offline` บนเว็บคิดจาก "เสียงหัวใจ" (heartbeat) ที่เอเจนต์ส่งมา — เงียบเกินกำหนด = offline = แค่โปรแกรมไม่ได้รัน
- เปิดโปรแกรมค้างไว้ระหว่างเล่น ปิดเกม/ปิดโปรแกรมเมื่อไรก็หยุดส่ง

---

## 1. สิ่งที่ต้องมีบนเครื่องใหม่

| ของ | หมายเหตุ |
|---|---|
| Windows | เกม TBH รันบน Windows |
| เกม **TBH: Taskbar Hero** | ติดตั้งและ **เปิดอย่างน้อย 1 ครั้ง** เพื่อให้มีไฟล์เซฟ |
| **Python 3.11+** | ดาวน์โหลดจาก python.org ตอนติดตั้ง **ติ๊ก "Add Python to PATH"** ให้ด้วย |
| Git (ไม่บังคับ) | ใช้ `git clone` ถ้าไม่มีก็โหลด ZIP จาก GitHub ได้ |

ตรวจว่าพร้อม:
```powershell
python --version      # ควรขึ้น 3.11 ขึ้นไป (บางเครื่องใช้คำสั่ง  py --version)
```

## 2. ⚠️ ที่อยู่เซิร์ฟเวอร์ (จุดพลาดอันดับ 1)

เอเจนต์ต้องรู้ว่า "เว็บอยู่ที่ URL ไหน" — **`http://localhost:3000` ใช้ได้เฉพาะบนเครื่องที่รันเว็บอยู่เท่านั้น** เครื่องอื่นเรียก `localhost` แล้วจะไม่เจอ

- **✅ ทางที่แนะนำ — เว็บขึ้น Vercel แล้ว ใช้ URL นี้ได้เลยจากทุกเครื่อง ทุกที่:**
  ```
  https://tbh-topaz.vercel.app
  ```
  ไม่ต้องเปิดไฟร์วอลล์ ไม่ต้องรันเว็บค้างบนเครื่องไหน — ข้ามไปข้อ 3 ได้เลย
- ทางสำรอง (ไม่จำเป็นถ้าใช้ Vercel ด้านบนแล้ว):
  - **เน็ตวงเดียวกับเครื่องที่รันเว็บ:** บนเครื่องที่รันเว็บสั่ง `pnpm dev -- -H 0.0.0.0`, เปิดพอร์ต 3000 ใน Windows Firewall, หา IP ด้วย `ipconfig` (`IPv4 Address` เช่น `192.168.1.50`) แล้วเครื่องใหม่ใช้ `http://192.168.1.50:3000`
  - **เครื่องใหม่รันเว็บเอง:** ใช้ `http://localhost:3000` บนเครื่องนั้นได้เลย

> จำ URL ที่เลือกไว้ จะต้องใส่ตอนข้อ 6 (ตอน setup)

## 3. เอาโค้ดลงเครื่อง

```powershell
# แนะนำ HTTPS (เลี่ยงปัญหา SSH port 22 โดนบล็อก)
git clone https://github.com/skrttboy420/TBH.git
cd TBH\agent
```

ถ้าไม่มี Git: เข้า GitHub repo → ปุ่ม **Code → Download ZIP** → แตกไฟล์ → เข้าโฟลเดอร์ `agent`
(จะรันเอเจนต์ ใช้แค่โฟลเดอร์ `agent/` พอ ไม่ต้องลงส่วนเว็บ)

---

## 4. ติดตั้งไลบรารี Python

```powershell
python -m pip install -r requirements.txt
```

ลงให้: `cryptography` (ถอดรหัสเซฟ), `requests` (คุยกับเว็บ), และของเสริม `Pillow` `pywin32` `keyboard` (แคปจอ/ปุ่มลัด)

---

## 5. สร้าง token ของเครื่องนี้บนเว็บ

1. เปิดเว็บแล้ว **ล็อกอินด้วยบัญชีตัวเอง**
2. ไปหน้า **Agents** (URL `/agents`)
3. กดปุ่ม **“เพิ่มเครื่อง”**
4. ตั้งชื่อเครื่องให้จำง่าย เช่น `พีซีห้องนอน` แล้วกด **“สร้าง”**
5. จะมีกล่อง **“เครื่องพร้อมใช้งาน”** โชว์ token ขึ้นต้นด้วย `tbh_…`
   👉 **คัดลอกเก็บไว้ทันที — โทเค็นแสดงแค่ครั้งเดียว!** (ถ้าพลาด กดเมนู ⋮ ของเครื่องนั้น → “หมุนโทเค็นใหม่” เอาอันใหม่)

---

## 6. ตั้งค่าเอเจนต์

```powershell
python run.py setup
```

ตอบทีละข้อ:

| ถาม | ตอบ |
|---|---|
| URL เซิร์ฟเวอร์ | URL จากข้อ 2 (แนะนำ `https://tbh-topaz.vercel.app`) |
| Agent Token | token `tbh_…` จากข้อ 5 |
| พาธไฟล์เซฟ | **เว้นว่าง** = ค้นหาอัตโนมัติ (ปกติเจอเอง) |
| ชื่อหน้าต่างเกม | `TaskBarHero` |
| ปุ่มลัด เริ่มฟาร์ม / หยุดฟาร์ม | `f6` / `f7` (หรือกด Enter ใช้ค่าเดิม) |

จบแล้วมันจะ **ทดสอบการเชื่อมต่อให้** — ขึ้น `✓ เชื่อมต่อสำเร็จ — เอเจนต์: <ชื่อ>` คือพร้อม
(พาธไฟล์เซฟปริยายบน Windows: `%USERPROFILE%\AppData\LocalLow\TesseractStudio\TaskbarHero\SaveFile_Live.es3`)

## 7. รันเอเจนต์

```powershell
python run.py
```

เปิดหน้าต่างนี้ค้างไว้ จะเห็น log ทำนองนี้ไหลเรื่อย ๆ:
```
[..] TBH Companion Agent v0.1.0
[..] เซิร์ฟเวอร์: https://tbh-topaz.vercel.app
[..] ส่งเซฟแล้ว · ทอง=920,507 · ของดรอป=0 · กิจกรรม=1
```
แล้วบนเว็บหน้า **Agents** เครื่องนี้จะกลายเป็น **online/farming** และประวัติ/ของดรอปจะเริ่มขึ้น

> อยากปิดเทอร์มินัลแต่ให้ยังรัน: รันด้วย `pythonw run.py` หรือทำเป็น Task Scheduler / .bat ไว้เปิดตอนเข้า Windows

## 8. คำสั่งสรุป (cheat sheet)

```powershell
git clone https://github.com/skrttboy420/TBH.git
cd TBH\agent
python -m pip install -r requirements.txt
python run.py setup      # ใส่ URL + token (สร้างจากหน้า /agents → เพิ่มเครื่อง)
python run.py            # เปิดค้างไว้ระหว่างเล่น
```

## 9. แก้ปัญหาที่เจอบ่อย

| อาการ | วิธีแก้ |
|---|---|
| `เชื่อมต่อเซิร์ฟเวอร์ไม่ได้` | URL ผิด / ใช้ `localhost` จากคนละเครื่อง / ไฟร์วอลล์บล็อกพอร์ต 3000 → ดูข้อ 2 |
| ภาษาไทยในเทอร์มินัลเพี้ยน | สั่ง `set PYTHONUTF8=1` ก่อนรัน หรือ `chcp 65001` |
| หาไฟล์เซฟไม่เจอ | ตอน setup ใส่พาธเต็ม: `C:\Users\<ชื่อ>\AppData\LocalLow\TesseractStudio\TaskbarHero\SaveFile_Live.es3` |
| `PermissionError` ตอนเกมกำลังเซฟ | ปกติ — เอเจนต์ลองอ่านใหม่เอง ไม่ต้องทำอะไร |
| `git clone` ค้าง (SSH port 22) | ใช้ลิงก์ **HTTPS** ตามข้อ 3 แทน |
| `python`/`pip` ไม่เจอ | ติดตั้ง Python ใหม่แล้วติ๊ก **Add Python to PATH** (หรือใช้คำสั่ง `py` แทน `python`) |
| token หลุด/ลืม | หน้า Agents → เมนู ⋮ ของเครื่อง → **หมุนโทเค็นใหม่** → `python run.py setup` ใส่ token ใหม่ |
| เปลี่ยน URL/พอร์ตทีหลัง | แก้ในไฟล์ `agent/config.json` ได้ตรง ๆ (มีคีย์ `apiBaseUrl`, `agentToken`, `savePath`) |

## หมายเหตุไฟล์ config

`python run.py setup` จะเขียน **`agent/config.json`** ให้ — ไฟล์นี้ **ไม่ถูกอัปขึ้น Git** (มี token) แต่ละเครื่องมีของตัวเอง ตัวอย่างหน้าตา:

```json
{
  "apiBaseUrl": "https://tbh-topaz.vercel.app",
  "agentToken": "tbh_xxxxxxxxxxxxxxxx",
  "savePath": "",
  "windowTitle": "TaskBarHero",
  "pollIntervalSeconds": 5,
  "heartbeatIntervalSeconds": 10,
  "controls": { "startFarmHotkey": "f6", "stopFarmHotkey": "f7" }
}
```
